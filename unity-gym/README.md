### 設定Unity-MCP 
Run Unity-6.3 LTS (6000.3.13f1)<br>
* Unity-Hub> Projects > [+ New project]<br>
* Window > Package Management> Package Manager> + > Install package from git URL... `https://github.com/CoplayDev/unity-mcp.git?path=/MCPForUnity#main`<br>
* Window > MCP For Unity > Toggle MCP Window<br>
